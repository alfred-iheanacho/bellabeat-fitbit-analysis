Bellabeat Fitbit Analysis

End-to-end health analytics project using Python, Pandas, NumPy, Matplotlib, and Seaborn to analyze Fitbit user behavior.

View GitHub Repository

Project Overview

This case study investigates Fitbit user activity, sleep behavior, and calorie expenditure to generate actionable wellness insights for Bellabeat.

The project covers the complete analytics workflow:

  • Data Cleaning
  • Feature Engineering
  • Data Quality Validation
  • Dataset Integration
  • Exploratory Data Analysis
  • Business Recommendations
